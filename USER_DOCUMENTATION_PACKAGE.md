## 📋 COMMON TASKS

### Installation
```
1. Go to: OutlookPhishingReporterSetup\Release\
2. Double-click: Install.vbs
3. Enter email: security@company.com
4. Restart Outlook
⏱️ Time: 2-3 minutes
```

### Fix Configuration Error
```
1. Go to: OutlookPhishingReporterSetup\Release\
2. Right-click: Diagnose-and-Repair.bat
3. Select: Run as Administrator
4. Enter email
5. Restart Outlook
⏱️ Time: 2-3 minutes
```

### Report Phishing Email
```
1. Select email in inbox
2. Click "Report phishing" button
3. Click OK to confirm
4. Email moves to Deleted Items
⏱️ Time: 30 seconds
```

### Uninstall
```
1. Go to: OutlookPhishingReporterSetup\Release\
2. Double-click: Uninstall.vbs
3. Click YES to confirm
4. Done!
⏱️ Time: 1 minute
```

### Change Security Email
```
1. Run: QuickFix.bat or Diagnose-and-Repair.bat
2. Enter new email
3. Restart Outlook
⏱️ Time: 2 minutes
```
